# FuncLab · 函数图像实验室

为学习高等数学打造的本地函数绘图软件，界面复刻 MATLAB 一类工程软件的风格。
纯 HTML/CSS/JavaScript 实现，**零依赖、无需安装、无需联网**。

## 快速开始

双击打开 [index.html](index.html) 即可（推荐 Edge / Chrome）。

函数列表与视图会自动保存在浏览器本地，刷新后恢复。

## 功能一览

| 模块 | 内容 |
|------|------|
| 绘图类型（11 种） | 显式函数 y=f(x) · 参数方程 · 极坐标 r(θ) · 隐函数 F(x,y)=0 · 数列 a(n) · **级数部分和 Σuₙ(x)** · **方向场 y′=f(x,y)** · **向量场 (P,Q)** · **梯度场 ∇f** · 三维曲面 z=f(x,y) · 空间曲线 |
| 参数滑块 | 表达式中的未知单字母（如 `a*sin(b*x)` 的 a、b）自动生成滑块，可设范围、可 ▶ 播放动画；适用于全部绘图类型，分析工具同步代入 |
| 分析工具 | 切线/法线 · 导数曲线 f′ f″（符号求导）· 定积分（区域着色 + Simpson + **黎曼和矩形动画**）· 泰勒展开（任意点、1–10 阶）· **特征点自动标注（零点/极值/拐点）** |
| 动态演示 | 级数部分和逐项累加动画（幂级数、傅里叶方波…）· 方向场过初值点的 RK4 积分曲线 · 梯度场等值线正交叠加 · 黎曼和 n→∞ 演示 |
| 示例库 | 摆线、心形线、玫瑰线、双纽线、圆锥曲线、重要极限、利萨茹、Logistic 方向场、e^x 幂级数逼近、马鞍面…… 共 9 类 40+ 经典图像 |
| 交互 | 滚轮缩放（Shift/Ctrl 单轴）· 拖动平移 / 3D 旋转 · 单击曲线显示坐标 · 等比例/网格/图例开关 · 导出 PNG |
| 表达式 | 隐式乘法（2x、3sin(x)、xy）、^ 乘方、! 阶乘、π/θ 直接输入、完整初等函数库，详见软件内「语法帮助」 |

## 架构

```
FuncLab/
├── index.html              入口与界面骨架
├── css/styles.css          MATLAB 风格主题（Ribbon 工具条 / 面板 / 对话框）
├── js/
│   ├── core/               ── 数学核心层（纯函数，无 DOM，可独立测试）
│   │   ├── runtime.js      内置函数注册表、常量、Γ 函数、数字格式化
│   │   ├── parser.js       词法 + 递归下降语法分析 → AST（隐式乘法、θ/π）
│   │   ├── compile.js      AST → 嵌套闭包（密集采样零解析开销）
│   │   ├── derivative.js   符号求导（链式法则）+ 化简（常量折叠等）
│   │   └── calculus.js     Simpson 积分、黎曼和、切线、泰勒展开、
│   │                       RK4 积分曲线、扫描求根（零点/极值/拐点）
│   ├── plot/               ── 渲染层（Canvas 2D，只读 core）
│   │   ├── viewport.js     世界↔屏幕坐标变换、缩放平移、1-2-5 刻度
│   │   ├── axes.js         boxed 坐标系：网格/零轴/边框/外侧刻度标签
│   │   ├── curves2d.js     显式/参数/极坐标/数列 采样与断点检测
│   │   ├── implicit.js     Marching Squares 隐函数等值线
│   │   ├── fields.js       方向场短斜线、向量场箭头、梯度场（箭头+等值线）、
│   │   │                   过初值点的 RK4 积分曲线
│   │   ├── surface3d.js    正交投影 + 画家算法 + parula 色图 + 坐标箱
│   │   └── plotter.js      渲染调度：曲线/场 → 工具叠加层 → 图例 → 数据提示
│   └── app/                ── 应用层（状态与 UI）
│       ├── state.js        类型元数据、函数条目编译、自由参数识别与绑定、
│       │                   级数部分和/梯度编译、持久化、示例加载
│       ├── examples.js     示例库数据（高数经典图像 + 动态演示）
│       ├── tools.js        分析工具的计算与面板 UI（含黎曼和、特征点）
│       ├── panel.js        函数卡片列表、添加菜单、参数滑块与动画
│       └── main.js         启动装配、Ribbon、画布交互、状态栏、导出
└── tests/
    ├── core.test.js        数学核心层单元测试（62 项）
    └── run-tests.html      浏览器测试页
```

依赖方向严格单向：`app → plot → core`。core 层不接触 DOM，
因此测试可直接在 node 下运行：

```
node tests/core.test.js
```

### 数据流

```
输入表达式 ──parser──▶ AST ──compile──▶ 求值闭包
                        │
                        └─derivative──▶ 导函数 AST（切线/泰勒/导数曲线）

求值闭包 ──按类型采样──▶ 世界坐标点列 ──viewport──▶ 屏幕坐标 ──▶ Canvas
```

## 表达式语法速查

- 运算：`+ - * /`，乘方 `^`（`-x^2` 按 `-(x^2)`），阶乘 `!`
- 可省略乘号：`2x`、`3sin(x)`、`(x+1)(x-1)`、`xy`
- 函数：`sin cos tan cot sec csc`、`asin/arcsin …`、`sinh …`、
  `exp ln log lg log2 log10`、`sqrt cbrt abs sign floor ceil round`、
  `min max mod gamma`
- 常量：`pi`（或 `π`）、`e`
- 隐函数可直接写等式：`x^2 + y^2 = 4`
- 极坐标变量可写 `θ`、`theta` 或 `t`
- 自由参数：未定义的单字母自动变成滑块，如 `a*sin(b*x)`、`a(1+cos(θ))`；
  连写 `ab` 按 a·b 拆分，`sinx` 这类疑似函数拼写会报错提示

## 已知边界与扩展方向

- 暂不支持分段函数 if/分支语法（可用 `abs/sign/floor` 组合近似）
- 泰勒展开依赖符号求导，`gamma/min/max` 等函数不可展开（会给出提示）
- 特征点扫描基于可见范围采样 + 二分精化，间距小于视宽 1/1400 的相邻特征点可能漏检
- 规划中：分段函数语法、极坐标/参数方程的弧长与面积工具、旋转体体积演示
