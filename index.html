绘图类型

类型输入示例
显式函数y = f(x)sin(x)/x，e^(-x^2)
参数方程x(t), y(t) 与 t 范围摆线 x = t - sin(t)，y = 1 - cos(t)
极坐标r(θ) 与 θ 范围心形线 2(1 + cos(θ))（变量可写 θ / theta / t）
隐函数F(x, y) = 0 或直接写等式x^2 + y^2 = 4，x^3 + y^3 - 3x*y = 0
数列a(n) 与 n 范围(1 + 1/n)^n
级数部分和通项 uₙ(x)（含 n，可含 x）与 n 范围x^n/n! → 部分和逼近 e^x，拖动/播放 N
方向场y′ = f(x, y)，可选初值点x - y，过 (0, 0) 自动画积分曲线（RK4）
向量场P(x, y) 与 Q(x, y)旋转场 P = -y，Q = x
梯度场f(x, y)，自动求 ∇f 并叠加等值线x^2 + y^2（箭头与等值线处处正交）
三维曲面z = f(x, y) 与 x、y 范围马鞍面 (x^2 - y^2)/4
空间曲线x(t), y(t), z(t)螺旋线 2cos(t), 2sin(t), t/3

参数滑块

表达式中可直接使用未定义的单字母参数（如 a·sin(b·x) 中的 a、b）， 函数卡片会自动生成对应滑块：拖动实时改变图像，点 ▶ 在范围内往返动画。 参数适用于所有绘图类型（显式、极坐标、隐函数、曲面……），切线、泰勒展开等分析工具也会代入当前参数值。 注意：连写 ab 按 a·b 理解；sinx 这类疑似函数拼写会提示改成 sin(x)。

分析工具说明

分析工具针对「显式函数 y = f(x)」：切线/法线、导数曲线（f′、f″，符号求导）、 定积分（区域着色 + Simpson 数值积分，可叠加黎曼和矩形：左/中/右端点取样，拖动或播放 n 观察矩形和逼近定积分）、 泰勒展开（在 x₀ 处展开到指定阶数并叠加图像）、 特征点（自动标注可见范围内的零点 ○、极大 ▲、极小 ▼、拐点 ◆，并列出坐标）。
